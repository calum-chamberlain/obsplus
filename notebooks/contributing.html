Contributing¶

Contributions to ObsPlus are welcomed and appreciated.

Here are a few details you will need:

  1. We are planning to use this git branching model (or something similar) starting on version 0.1.0 when ObsPlus is expected to gain a bit more stability. Until then, we will just make feature commits and bug-fix commits directly to master and roll out releases as quickly as needed
  2. ObsPlus uses Black for code linting. If you install ObsPlus in developer mode it should install a pre-commit hook to run black.
  3. Use numpy style docstrings.
  4. ObsPlus makes extensive use of Python 3’s type hints. You are encouraged to annotate any public functions/methods with type hints.
